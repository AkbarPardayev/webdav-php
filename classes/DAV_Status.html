$status

int

the HTTP/1.1 status code, defaults to DAV::HTTP_INTERNAL_SERVER_ERROR

$info

mixed

One of the following: - for a 3xx status: the URL of the Location: response header - for a 4xx status: either - an array of pre- or postconditions, with the conditions as keys and extra XML as values - a free error string - for a 5xx status: a free error string
