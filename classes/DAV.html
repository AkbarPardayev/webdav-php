Outputs HTTP/1.1 headers.

header(\$properties $properties) : void

see \global\status_code()

Parameters

$properties

\$properties

array|string An array of headers to print, e.g. array( 'Content-Language' => 'en-us' ) If there's a key «status» in the array, it is used for the 'HTTP/1.X ...' status header, e.g.array( 'status' => DAV::HTTP_CREATED, 'Content-Type' => 'text/plain' ) If $properties is a string, it is taken as the Content-Type, e.g.$rest->header('text/plain') is exactly equivalent to $rest->header( array( 'Content-Type' => 'text/plain' ) );
